Cara penggunaan aplikasi:

- Halaman Home

Read
1. Pada halaman Home, segala produk yang memiliki stok di atas 0 akan ditampilkan.
Grid and List
2. Pengguna dapat mengganti mode produk yang ditampilkan ke dalam bentuk List atau Grid dengan menekan icon button yang ada pada sisi kanan toolbar.
Detail
3. Dengan menekan salah satu item produk, pengguna akan diarahkan ke halaman detail yang menampilkan informasi detail dari produk tersebut berdasarkan jenisnya.
Carousel
4. Pada halaman detail, foto produk berada dalam bentuk carousel dengan menggunakan ion-slides, sehingga pengguna dapat menggeser foto ke kiri ataupun ke kanan untuk melihat foto produk lainnya.

- Halaman Admin

Read
1. Pada halaman Admin, seluruh produk akan ditampilkan.
Create
2. Dengan menekan icon button '+' pada sisi kanan toolbar, Admin akan diarahkan pada form tambah produk, dimana Admin dapat memasukkan informasi-informasi mengenai produk berdasarkan jenis produk yang dipilih.
Update dan Geser ke Kiri
3. Dengan menggeser salah satu item produk ke kiri, terdapat icon update yang jika Admin tekan akan diarahkan ke form update produk untuk memperbaharui data dari produk tersebut berdasarkan jenisnya.
4. Dalam pengisian form tambah produk dan form update produk, Admin harus mengisi seluruh input field dengan data yang valid agar button untuk menambah atau memperbaharui produk dapat ditekan.
Multi Delete dan Geser ke Kanan
5. Dengan menggeser salah satu item produk ke kanan, terdapat checkbox yang jika Admin centang akan menunjukkan bahwa produk tersebut masuk ke dalam list produk yang ingin dihapus.
6. Setelah memilih produk-produk yang ingin dihapus, Admin dapat menekan icon button delete (trash) yang ada pada sisi kanan toolbar untuk menghapus seluru produk yang telah dicentang.
7. Ketika Admin menekan icon button delete tersebut tanpa mencentang produk apapun, Toast akan muncul yang menunjukkan bahwa Admin harus mencentang setidaknya satu produk untuk menggunakan fitur delete tersebut.
Detail
8. Dengan menekan salah satu item produk, Admin akan diarahkan ke halaman detail yang menampilkan informasi detail dari produk tersebut berdasarkan jenisnya (fitur-fitur pada halaman detail Admin sama seperti halaman detail pengguna biasa).
